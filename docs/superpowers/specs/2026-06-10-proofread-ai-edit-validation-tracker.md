另一個 E2E 觀察（段 0 精簡）：`對我是 匹特殊的馬` → `係匹特殊馬` — 同樣 書面「是」→口語「係」。

**結論：「精簡」指令有系統性 register-drift pattern（書面語軌走咗去口語）→ 修 prompt。**

## 修正

`ai_edit.build_system_prompt` 規則 1 加語體保持：「輸出必須維持同原字幕一致嘅語言、書寫系統（繁／簡）**同語體（書面語定口語）**…亦唔好改變語體（除非指令明確要求翻譯或者改語氣）」。Commit：`fix(ai-edit): register-preserve prompt rule`。
